RateMyShop - Frontend Application

This is the user interface for the RateMyShop application built with React.

PREREQUISITES
1. Node.js must be installed.
2. The Backend server must be running on port 5000 before starting this.

SETUP INSTRUCTIONS

1. Open a new terminal window.
2. Navigate to the frontend folder:
   cd frontend
3. Install the required libraries:
   npm install
4. Start the application:
   npm start

The application will open in your browser at localhost:3000.

FEATURES

Login System:
Different dashboards for Admins, Users, and Store Owners.

Admin Dashboard:
Manage users and stores. View system-wide statistics.

User Dashboard:
Search for stores by name or address.
Submit ratings (1-5 stars) for stores.
Change password.

Owner Dashboard:
View average rating for your store.
See a list of customers who rated your store.
Change password.
